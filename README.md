# Herramientas de Inteligencia Artificial de Uso General

Este documento recopila un conjunto de herramientas de inteligencia
artificial diseñadas para apoyar distintos tipos de proyectos, desde
desarrollo de software hasta análisis y productividad. Estas soluciones
permiten optimizar flujos de trabajo, mejorar la calidad del código y
facilitar la interacción con documentación técnica.

------------------------------------------------------------------------

## 1. Terminal Inteligente

**Warp**\
Sitio web: https://www.warp.dev/terminal

Warp es una terminal moderna potenciada con inteligencia artificial que
mejora la experiencia del desarrollador mediante autocompletado
inteligente, ejecución de comandos optimizada y una interfaz más
intuitiva en comparación con terminales tradicionales.

------------------------------------------------------------------------

## 2. Lectores de Documentación Inteligentes

**DeepWiki**\
Sitio web: https://deepwiki.org/

DeepWiki es una plataforma desarrollada por Cognition que permite
indexar repositorios de GitHub y explorar su contenido mediante un chat
impulsado por inteligencia artificial (como Devin).

### Características principales:

-   Indexación automática de repositorios.
-   Exploración interactiva del código.
-   Comprensión contextual mediante chat.
-   Posibilidad de agregar nuevos repositorios para análisis.

------------------------------------------------------------------------

## 3. Skills (Habilidades Open Source)

**Skills.sh**\
Sitio web: https://skills.sh/

Skills.sh es una plataforma que permite utilizar y gestionar habilidades
(skills) open source orientadas a mejorar procesos de desarrollo
asistidos por inteligencia artificial.

### Ejemplo de instalación de un skill:

    npx skills add https://github.com/vercel-labs/agent-skills --skill vercel-react-best-practices

Este comando permite integrar buenas prácticas de desarrollo en React,
proporcionadas por el repositorio de Vercel.

------------------------------------------------------------------------

## 4. Code Review Automatizado

**Devin Review**

Devin Review es una herramienta de revisión automática de código basada
en inteligencia artificial.

### Características principales:

-   Revisión continua de código.
-   Identificación de errores y vulnerabilidades.
-   Sugerencias de mejora basadas en estándares.
-   Apoyo en la adopción de buenas prácticas de desarrollo.
-   Incremento de la productividad en equipos de desarrollo.
