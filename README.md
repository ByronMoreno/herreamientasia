# Herramientas de Inteligencia Artificial de Uso General

Este documento recopila un conjunto de herramientas de inteligencia
artificial diseñadas para apoyar distintos tipos de proyectos, desde
desarrollo de software hasta análisis y productividad. Estas soluciones
permiten optimizar flujos de trabajo, mejorar la calidad del código y
facilitar la interacción con documentación técnica.

------------------------------------------------------------------------

## 1. Terminal Inteligente

**Warp**\
Sitio web: https://www.warp.dev/terminal

Warp es una terminal moderna potenciada con inteligencia artificial que
mejora la experiencia del desarrollador mediante autocompletado
inteligente, ejecución de comandos optimizada y una interfaz más
intuitiva en comparación con terminales tradicionales.

------------------------------------------------------------------------

## 2. Lectores de Documentación Inteligentes

**DeepWiki**\
Sitio web: https://deepwiki.org/

DeepWiki es una plataforma desarrollada por Cognition que permite
indexar repositorios de GitHub y explorar su contenido mediante un chat
impulsado por inteligencia artificial (como Devin).

### Características principales:

-   Indexación automática de repositorios.
-   Exploración interactiva del código.
-   Comprensión contextual mediante chat.
-   Posibilidad de agregar nuevos repositorios para análisis.

------------------------------------------------------------------------

## 3. Skills (Habilidades Open Source)

**Skills.sh**\
Sitio web: https://skills.sh/

Skills.sh es una plataforma que permite utilizar y gestionar habilidades
(skills) open source orientadas a mejorar procesos de desarrollo
asistidos por inteligencia artificial.

### Ejemplo de instalación de un skill:

    npx skills add https://github.com/vercel-labs/agent-skills --skill vercel-react-best-practices

Este comando permite integrar buenas prácticas de desarrollo en React,
proporcionadas por el repositorio de Vercel.

------------------------------------------------------------------------

## 4. Code Review Automatizado

**Devin Review**

Devin Review es una herramienta de revisión automática de código basada
en inteligencia artificial.

### Características principales:

-   Revisión continua de código.
-   Identificación de errores y vulnerabilidades.
-   Sugerencias de mejora basadas en estándares.
-   Apoyo en la adopción de buenas prácticas de desarrollo.
-   Incremento de la productividad en equipos de desarrollo.

### Ejemplo de uso: Devin Review

Para utilizar Devin Review, se debe seguir un flujo básico de integración con repositorios.

#### Paso 1: Repositorio de origen (GitHub)
https://github.com/spring-projects/spring-petclinic

#### Paso 2: Importación en Devin Review
El repositorio se conecta o importa en la plataforma de Devin Review para su análisis.
https://devinriew.com/spring-projects/spring-petclinic

#### Paso 3: Solicitud de revisión

Una vez integrado, se puede solicitar una revisión automática del código con una instrucción como:

> "Revisa este repositorio y sugiere mejoras en calidad, seguridad y rendimiento."

#### Paso 4: Resultados del análisis

Devin Review generará observaciones como:

- Identificación de errores potenciales.
- Detección de malas prácticas.
- Recomendaciones de optimización.
- Sugerencias de mejora en seguridad.

#### Paso 5: Aplicación de mejoras

Finalmente, el desarrollador puede aplicar las recomendaciones para mejorar la calidad del software.

## 5. CodeRabbit

**coderabbit ia**

Sitio web: https://www.coderabbit.ai/

CodeRabbit es una herramienta de revisión de código con inteligencia artificial que analiza Pull Requests y sugiere mejoras automáticamente.

**Precio:**
- Gratis para repositorios públicos (open source).
- Prueba gratuita (trial) del plan completo por tiempo limitado.
- Plan Pro de pago para equipos y uso profesional (incluye revisiones avanzadas e integraciones).

------------------------------------------------------------------------

## 6. Vercel

Sitio web: https://vercel.com/

Vercel es una plataforma de despliegue y hosting que permite publicar aplicaciones web (especialmente con frameworks como React, Next.js) de forma rápida y automatizada.

**Precio:**
- Plan gratuito con despliegues básicos y proyectos personales.
- Prueba y uso inicial sin costo.
- Planes de pago para equipos, mayor rendimiento y funcionalidades avanzadas.

------------------------------------------------------------------------

## Kilo Code

Sitio web: https://kilocode.ai/

Kilo Code es una herramienta de inteligencia artificial enfocada en asistir el desarrollo de software, permitiendo generar, analizar y mejorar código de manera automática.

**Precio:**
- Plan gratuito con funcionalidades básicas.
- Prueba gratuita de características avanzadas.
- Planes de pago para uso profesional y mayor capacidad.
